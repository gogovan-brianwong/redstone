Web admin UI for Kubernetes cluster. 

Requirement:

Frontend:
  - bootstrap
  - jQuery
  - D3 Force

Backend:
  - Django
  - MySQL or Mariadb
  - PyMySQL
  - Jinja2
  - Kubernetes python lib ( install kubernetes via pip )
  - Paramiko
